# 汕大课程表小程序

# 概要
汕大课程表为汕头大学学生自主成立的非营利项目，是所有汕大学生都使用的软件。
# 功能
+ 同步课表信息
+ 自定义课表
+ 上课提醒

# 目录结构
+ image ---------------- 图片资源
+ pages 
    + classes ----------- 课程页面
    + me --------------- 个人页面
+ style ------------------ 样式
+ util ------------------- 工具包
+ app.js ---------------- 全局逻辑
+ app.json ------------- 全局配置
+ app.wxss ------------- 全局样式
